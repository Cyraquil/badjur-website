Extra stuff

This page is focus on extra stuff that cannot or not planned be a added the base modpack. But can be install by yourself to improved your experience. Some Extra can contain a pre-configured config file to match with the modpack. But please take not I do not offer support or help in case those extra dosen't work. But I can sometime update the config file in case the mod heavely modified and broke the presets in a update.


Xero's minimaps Pokémon radar

In the new release 1.0. The optional Xaero's minimap was removed to keep the consistency of the Pokenav and pokéradar items (Pokénav mod) and keeping the experience more accurate. But you can re-added Xaero's mods by your own if you wish. I included a configuration file preset for the modpack with the correct minimap position, displayed info and pokémon radar icons. In addition. Adding those mod as addon should not break when the modpack update as the modpack work now as mrpack file format.


*Make sure you select the Fabric version for 1.21.1 !*
- Xaero's Minimap (Mod) : https://modrinth.com/mod/xaeros-minimap
- Xaero's Worldmap (Mod) : https://modrinth.com/mod/xaeros-world-map
